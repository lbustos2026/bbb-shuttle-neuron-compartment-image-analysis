# bbb-shuttle-neuron-compartment-image-analysis
This repository contains an R pipeline for IN Carta-based analysis of BBB shuttle siRNA-Cy5 signal dynamics across cortical neuron subcellular compartments (EEA1, RAB11, LAMP1) over early timepoints.

This analysis generates **Figure 8D** from the associated manuscript.

## Overview ##
Systemically delivered BBB shuttle siRNA-Cy5 signal was quantified within NeuN+ neuron subcellular compartments:

- **EEA1+** (early endosomes)
- **RAB11+** (recycling endosomes)
- **LAMP1+** (lysosomes)

This pipeline processes object-level imaging exports and performs:

- Background-subtracted Cy5 intensity extraction
- Normalization to control (Group A)
- Detection of bimodal high-intensity subpopulations
- Per-well (animal-level) aggregation
- Group-level summary statistics
- Compartment-specific and combined visualization

## Data Structure ##
The input data comes from **IN Carta object-level exports**, organized by compartment and experimental group.
After unzipping the data, the structure should look like:
```
Data/
├── NeuNEEA1Cy5-HPRT/
│ ├── A/
│ ├── G/
│ ├── H/
│ ├── I/
│ ├── J/
│ └── K/
├── NeuNRAB11Cy5-HPRT/
│ ├── A/
│ ├── G/
│ ├── H/
│ ├── I/
│ ├── J/
│ └── K/
└── NeuNLAMP1Cy5-HPRT/
├── A/
├── G/
├── H/
├── I/
├── J/
└── K/
```
Each group folder contains IN Carta exports such as:
- `*_ObjectData.csv` ← **used in this analysis**
- `*_FieldData.csv`
- `*_WellData.csv`

**Only the `ObjectData` files are used by this pipeline**

### Running the Analysis ###
1. Clone or download this repository
3. Open R and ensure the Data/ folder is present in the `project_dir`
4. Run Script: "BBB_Shuttle_Neuron_Image_AnalysisScript_LBustos_2026.R"

### Output ###
Outputs/
├── EE/
├── RE/
├── Lyso/
└── Compartment_Analysis/
 Generated outputs include:

1. Summary tables
- Per-well (animal-level) normalized Cy5 intensity
- Tail vs. main population statistics
- Group-level summaries

2. Plots
- Individual compartment distributions (EEA1, RAB11, LAMP1)
- Tail-filtered distributions
- Smoothed trajectory plots (Figure 8D)
   
### Analysis Notes ###
Signal is computed from:
- Mean intensity (Cy5) with local background subtraction (BG3)
Data is aggregated:
- object → well (animal) → group
Normalization:
- relative to Group A (Naive)
Visualization:
- LOESS-smoothed mean ± SD

### NOTES ###
- Only ObjectData files are required
- Folder names (A, G, H, I, J, K) must be preserved
- The script assumes IN Carta naming conventions
